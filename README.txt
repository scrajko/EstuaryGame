

main() is in src/estuary/controller/Controller.java

Main Menu:
    Click to start

Maze Game:
    Use arrow keys to move Crab

Horseshoe Game:
    Use mouse to direct light
	Use mouse to click on and count horseshoe crabs
	
Shore Game:
    Use arrow keys to move Crab
	Use W key to place a Wall on the shore
	Use G key to construct an Oyster Gabion to hold
	Use G key again to place said Oyster Gabion on shore
	
End Screen:
	Click to restart to main menu

Use Ctrl + S to save game state to game.sav file
Use Ctrl + L to load game state from game.sav file
